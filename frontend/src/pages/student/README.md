# Ensias-Eats

Ensias-Eats is a comprehensive food management platform designed for students. It allows users to order meals, track their nutritional intake, manage a digital wallet, and view their order history.

## Features

- **Dashboard**: Overview of wallet balance, calories consumed, and active orders.
- **Meals**: Browse menus, filter by category, and add items to the cart.
- **Orders**: Track order status (Pending, Ready, Completed) and view QR codes for pickup.
- **Wallet**: Manage funds, recharge via QR code, and view transaction history.
- **Food Diary**: Log meals from the cafeteria or external sources to track nutrition.
- **Profile**: Manage personal settings and nutritional goals.

## Getting Started

### Prerequisites

- Node.js installed

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
